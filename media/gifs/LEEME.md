# GIFs de señas

Este directorio se sirve como contenido estático; la base de datos guarda
las rutas relativas a esta carpeta (p. ej. `abecedario/a.gif`), nunca los
binarios.

## ⚠️ Estado actual: PLACEHOLDERS de desarrollo

Los GIFs de `abecedario/` son **placeholders generados** por
`db/seeds/generar_gifs_placeholder.py`: muestran la letra con la marca
parpadeante «PLACEHOLDER · no es la seña real». **No representan señas
reales de la Lengua de Señas Peruana.**

El contenido real (GIFs de señas y descripciones de ejecución) se
incorporará mediante el módulo de importación (`backend/importador/`),
sin modificar el código fuente (RF11).
